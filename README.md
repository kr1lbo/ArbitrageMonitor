# Crypto Price Monitor

Мини-инструмент для отслеживания цен криптовалют в реальном времени через WebSocket-подключения бирж.
Поддерживает **CLI** и **GUI (PyQt5)** режимы.

---

## 🚀 Возможности

* Мониторинг выбранного токена на нескольких биржах одновременно.
* Реальное время: цена, 24h объём, время обновления.
* Расчёт **дельты (%)** с момента запуска мониторинга.
* Автоматическое определение доступности пары на бирже.
* Открытие каждого токена в **отдельном окне** (опционально).
* Тёмный аккуратный интерфейс.
* Полностью асинхронное получение данных через **ccxt.pro**.

---

## 📦 Требования

```
Python 3.10+
ccxtpro

```



---

## ▶ Запуск

Приложение запускается из `main.py`.

```bash
python main.py
```

После запуска будет предложено:

```
1 - CLI
2 - GUI
Enter - Exit
```

### **CLI-режим**

Информация выводится в терминал в виде таблицы.

### **GUI-режим**

Откроется окно:

* поле ввода токена (например: `BTC/USDT`);
* кнопка **Открыть** — запускает мониторинг;
* кнопка **Настройки** — выбор бирж и режима открытия (в новом окне или текущем).

Каждый монитор открывается компактным окном с таблицей обновлений.
