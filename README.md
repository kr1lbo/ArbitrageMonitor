# ArbitrageMonitor

Десктопный монитор спредов между криптовалютными биржами в реальном времени.

## Что делает

- Подключается к биржам через WebSocket и получает цены в реальном времени
- Показывает спреды между всеми парами бирж (SPOT → PERP арбитраж)
- Отображает фандинг по каждой стороне и итоговый Fund Result
- Звуковой сигнал при достижении заданного спреда

## Интерфейс

**Левая панель — таблица спредов:**

| Колонка | Описание |
|---|---|
| КУПИТЬ (LONG) | Биржа с низкой ценой + цена |
| ПРОДАТЬ (SHORT) | Биржа с высокой ценой + цена |
| СПРЕД % | Разница цен в процентах |
| FUND LONG | Фандинг на long-стороне. Зелёный = получаем |
| FUND SHORT | Фандинг на short-стороне. Зелёный = получаем |
| FUND RESULT | Чистый P&L по фандингу за период |

**Правая панель** — статус подключения по каждому источнику.

**Управление таблицей:**
- Клик на заголовок колонки — сортировка (повторный клик меняет направление)
- Drag & drop строк — ручная перестановка пар в удобном порядке

## Настройки

Открываются кнопкой ⚙ Настройки:
- Включение/отключение бирж
- Максимальное количество строк в таблице
- Сигнальный спред — порог для звукового сигнала
- Звуковой файл — mp3 или wav

Настройки сохраняются в `config.json` и применяются при следующем запуске.

## Логика фандинга

```
FUND LONG  < 0 → зелёный  (на long получаем фандинг)
FUND LONG  > 0 → красный  (на long платим фандинг)
FUND SHORT > 0 → зелёный  (на short получаем фандинг)
FUND SHORT < 0 → красный  (на short платим фандинг)

FUND RESULT = -FUND_LONG + FUND_SHORT
```

Пары с `FUND RESULT ≥ 0.20%` показываются даже при маленьком спреде.
